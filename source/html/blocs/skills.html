  • HTML5

    Каркас сайта, скелет. Делаем валидным и семантическим. Валидность — «уважают» другие веб-разработчики, а семантику «уважают» поисковики.

  • CSS3

    Язык стилей, который постоянно развивается, совершенствуется. По факту это внешний вид сайта, его одежа. Графика и геометрия. Тут полет фантазий.

  • Java Script

    Язык программирования, благодаря которому можно «оживить» что угодно: слайдеры, окна, подсказки, курсор мышки, вкладки, получение данных с сервера, расчеты и вычисления и многое другое.

  • Jquery

    Библиотека JS для ускорения разработки. Навык работы с JQuery присутствует, но я все же предпочитаю использовать нативный JS.

  • Figma

    Инструмент веб-дизайнера для создания прототипов сайтов, интерфейсов приложений. В этом графическом онлайн-редакторе обсуждаем правки и предложения по макету.

  • Wordpress

    Популярная CMS. Использую для создания панели управления, что позволит управлять сайтом самостоятельно, наполнять контентом, менять картинки, без участия разработчика.

  • Gulp

    Инструмент-сборщик, который помогает автоматизировать рутинные задачи в процессе разработки. Владеешь Gulp`ом, работаешь в разы быстрее. Это как «заточить пилу».

  • Sass

    Использую для «модульного» подхода в работе с CSS. В конечном варианте нечего не меняется, но работа получается комфортнее и быстрее. Легче читать и поддерживать код.

  • Avocode

    Это приложение для работы с файлами .psd (photoshop) и другим популярным у дизайнеров редакторе Sketch. Работаем с кодом, считываем макет, делаем верстку. Удобно? Удобно.

  • GIT | GitHub

    Инструмент записи изменений проекта, в процессе работы. Если возникают ошибки, возможно вернуть проект на любую стадию или рабочее состояние «до».

  • Bootstrap

    Фреймворк, включает в себя готовые шаблоны для создания сайтов и веб-приложений. Но, несмотря на все это удобство, предпочитаю верстать нативно, без шаблонов.
