Навыки

Стек технологий

  • HTML5

    Разметка, каркас сайта, скелет. Валидная разметка является легко читаемой другими веб-разработчиками и является более дружественной для поисковиков.

  • CSS3

    Язык стилей, который постоянно развивается, совершенствуется. По факту это внешний вид сайта, его одежа. Графика и геометрия. Тут полет фантазий.

  • Java Script

    Язык программирования, благодаря которому можно «оживить» что угодно: слайдеры, окна, подсказки, курсор мышки, вкладки, получение данных с сервера и многое другое.

  • Jquery

    Библиотека JS которая позволит ускорить разработку. Навык работы с ней присутствует, но я предпочитаю использовать нативный JS, так сайт работает быстрее.

  • Figma

    Инструмент веб-дизайнера для создания прототипов сайтов, интерфейсов приложений. В этом графическом онлайн-редакторе можем обсуждать правки макета в режиме реального времени.

  • Wordpress

    Популярная CMS. Использую для создания панели управления, что позволит управлять сайтом самостоятельно, наполнять контентом, менять картинки, без участия разработчика.

  • Gulp

    Инструмент-сборщик, который помогает автоматизировать рутинные задачи в процессе разработки. Владеешь Gulpом, работаешь в разы быстрее. Это как «заточить пилу».

  • Sass

    Используется для упрощения файлов CSS. В конечном варианте нечего не меняется, но работа получается комфортнее и быстрее. Легче читать и поддерживать код.

  • Avocode

    Это приложение для работы с файлами Photoshop и другим популярным у дизайнеров редакторе Sketch. Работаем с кодом, считываем макет, делаем верстку. Удобно? Удобно.

  • GIT | GitHub

    Система контроля версий. Некое хранилище, где лежат файлы проекта. Благодаря этой системе, можно легко вернуть проект в состояние, когда он был согласован или вообще работал.

  • Bootstrap

    Фреймворк, включает в себя готовые шаблоны для создания сайтов и веб-приложений. Но, несмотря на все это удобство, предпочитаю верстать нативно, без шаблонов.
